Unit Price Checker

Compare groceries, drinks, and household items by actual cost per unit. Great for the classic store aisle question: is the giant one genuinely a better deal, or is the shelf tag lying to me again?

  • Use pack count for deals like 3 for $5.
  • Add a coupon to see the real checkout price.
  • Set a target amount to see the cheapest total for what you actually need.

    Quick tips

    • For a sale like 3 for $5, set price to $5.00 and pack count to 3.
    • You can compare 32 oz against 2 lb without doing conversion math in your head like it’s 2007.
    • Use target amount when you need a specific quantity and want the cheapest actual checkout total, not just the best unit price.

    Why it’s useful

    Simple enough to use in a parking lot, useful enough to keep around.

    Mixed-unit conversion

    Compare ounces to pounds, milliliters to liters, or count-based items without grabbing a second calculator.

    Sale-aware pricing

    Pack counts and coupons are built in, so the result reflects what you actually pay instead of the fantasy sticker price.

    Need-based totals

    See which option is cheapest for the amount you actually need, because the best unit price does not always win the real checkout.

    Shareable state

    Copy a link with the current comparison baked in. Handy for sending “which one should I buy?” debates to somebody else.
